API Reference

Base URL: http://localhost:8200 (default)

The Kalypso API uses plain HTTP/JSON with no authentication. It is designed for local dev networks only.

GET /health

Health check and statistics. Useful for Docker HEALTHCHECK and monitoring.

Response

{
  "status": "ok",
  "ca_initialized": true,
  "issued_count": 42
}

GET /ca.pem

Download the root CA certificate as raw PEM. Save directly to a file — no JSON parsing needed.

Response

Content-Type: application/x-pem-file

-----BEGIN CERTIFICATE-----
MIICxjCCAk2gAwIBAgIU...
-----END CERTIFICATE-----

Usage

curl http://localhost:8200/ca.pem > kalypso-ca.pem

GET /ca.json

CA certificate with metadata as JSON.

Response

{
  "certificate": "-----BEGIN CERTIFICATE-----\n...",
  "fingerprint": "ab:cd:ef:...",
  "not_before": "2025-01-01T00:00:00",
  "not_after": "2035-01-01T00:00:00",
  "subject": "CN=Kalypso Dev CA Root CA,O=Kalypso Dev CA"
}

POST /certificates

Issue a new short-lived SSL certificate.

Request Body

FieldTypeRequiredDefaultDescription
domainsstring[]Yes—Domain names for the certificate
hoursintNo24Lifetime in hours (1–168)
ip_addressesstring[]No[]IP SANs to include

Example Request

{
  "domains": ["myapp.local", "*.myapp.local"],
  "hours": 24,
  "ip_addresses": ["127.0.0.1"]
}

Response

{
  "certificate": "-----BEGIN CERTIFICATE-----\n...",
  "private_key": "-----BEGIN PRIVATE KEY-----\n...",
  "domains": ["myapp.local", "*.myapp.local"],
  "not_after": "2025-01-02T12:00:00Z",
  "ca_certificate": "-----BEGIN CERTIFICATE-----\n..."
}

Error Responses

  • 422 — Invalid request (empty domains, invalid hours, etc.)

OpenAPI / Swagger

When the server is running, visit these URLs for interactive API documentation:

  • Swagger UI: http://localhost:8200/docs
  • ReDoc: http://localhost:8200/redoc

Security Considerations

  • The API has no authentication — it's designed for local dev only
  • Never expose port 8200 to the public internet
  • The CA private key is stored at /data/ca-key.pem inside the container
  • All issued certificates are short-lived (max 7 days)
